# SQL Eğitimi Mini Projesi

Bu projede, SQL dilinin temel ve ileri seviye konularını kapsayan örnekler yer almaktadır. Veritabanı yönetimi konularına dair birçok yapıyı uygulamalı olarak gösteren mini bir çalışmadır.

## 📚 İçerik

- **Trigger:** Otomatik kayıt işlemleri ve veri denetimi için örnek trigger fonksiyonları
- **View:** Karmaşık sorguların sadeleştirilmiş halleri
- **Function:** Tekrar kullanılabilir SQL fonksiyonları
- **Join:** Tablo birleştirme örnekleri (INNER, LEFT, RIGHT JOIN)
- **Veri Girişi ve Örnek Kayıtlar**

## 🎯 Amaç

SQL öğrenim sürecimi pekiştirmek ve gerçek projelerde kullanabileceğim yapıların mantığını kavramak.

## 🛠 Kullanım

1. Veritabanı yönetim sisteminizde (`PostgreSQL`, `MySQL` vb.) proje dosyalarını çalıştırın.
2. `01_create_tables.sql` dosyasıyla tablo yapısını oluşturun.
3. `02_insert_data.sql` dosyasıyla veri girişlerini yapın.
4. `03_join_wherw_select_queries.sql`, `04_views.sql`, `05_functions.sql`	ve `06_triggers.sql` gibi dosyaları sırasıyla çalıştırarak örnek yapıları test edin.
